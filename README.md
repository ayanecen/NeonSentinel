# Neon Sentinel

「フロントエンドの縦スクロール固定砲台STG」

## ゲーム概要
Neon Sentinelは、ネオン調のグラフィックが特徴的な縦スクロール・シューティングゲームです。
画面下部の砲台を操作し、上から迫りくる敵を撃破してください。GitHub Pagesですぐにプレイ可能です。

## 操作方法
### PC
- **移動**: `A` / `D` または `←` / `→`
- **発射**: `SPACE` キー または マウス左クリック（押しっぱなしで連射可能）

### モバイル
- **移動・発射**: 画面をタップ、またはスライドして操作（タップ中は自動で連射されます）

## 仕様・特徴
- **難易度設定**: 「Easy」「Normal」「Hard」の3段階。難易度でライフ、敵の湧き、速度、スコア倍率が変化。
- **プロシージャルな敵**: 頂点ごとに半径をランダム化した不定形ポリゴンを生成。
- **レアUFO**: 出現率5%。通常の敵より硬い（HP2）が、撃破で高得点。
- **ネオン演出**:
  - 画面全体のグロー効果（Canvas Shadow）
  - スクロールするネオングリッド背景
  - 被弾時のスクリーンシェイク
  - UFO撃破時のフラッシュ演出
  - 爆発パーティクル
- **サウンド**: Web Audio APIによるシンセサイズド・サウンド（外部音源不要、ON/OFF可能）。

## 技術構成
- `index.html`: UIレイヤーとp5.jsの読み込み。
- `style.css`: ネオン/サイバーパンクな見た目のスタイリング。
- `script.js`: ゲームロジック、描画、音声制御（クラスベースのクリーンな設計）。

## 開発者向け
- 依存ライブラリ: [p5.js](https://p5js.org/) (CDN経由)
- 実行方法: `index.html` をブラウザで開く。
- パフォーマンス: 60fpsをターゲット。オブジェクトの寿命管理を徹底。
